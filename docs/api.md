# API Documentation

## Base URL

/api/v1

## Auth

### POST /auth/signup

Create new user

### POST /auth/login

Authenticate user

---

## User

### GET /users/me

Get current user

---

## Chat

### POST /chats

Create chat

### GET /chats

Get user chats

---

## Messages

### GET /messages/:chatId

Fetch messages

---

## Media

### POST /media/upload

Upload file

---

## Standards

* JSON responses
* JWT authentication
* Proper HTTP status codes
